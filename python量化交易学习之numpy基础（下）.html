Python量化交易学习之numpy基础（下）

这是学习股票量化工具numpy的最后一篇，准备把原书所讨论的正态分布复习一遍；而关于伯努利分布的内容我认为其并没有引入关于numpy的新知识，只是为了说明有手续费没有有效内幕消息的股票市场市场与有抽头没老千的赌场类似，如果只注重输赢而不注重盈多亏少，必定招致极大的亏损。

关于正态分布

特点：

  • 数据呈正态分布时，标准差越大，离散程度越大；
  • 数据呈正态分布时，期望值位于曲线对称轴的中心。

本章缺点：

  • 超纲内容：scipy.stats.norm.pdf()
  • 为什么要生成股票0的最大值与最小值之间的等距切分数据？
  • 未说明内容：np.cumsum()

关于正态分布策略

此处作者提出了新的两个numpy方法：

  • np.sort() 对序列进行排序
  • np.argsort() 将展示排序的原序列号，在此例中，原序列号就意味着对应的涨跌数据的股票代码

原书策略思路：

  1. 将后50天用来验证策略的数据，前454天用来得出策略（我认为从原始数据中随机分离出一半数量的数据作为测试集更好，预测用的模型不能对过去和将来两个时间段有任何歧视，否则历史数据将只能代表过去，而不能总结出规律指导未来；可以认为这是一种不典型的过拟合）；
  2. 打印出前454天中跌幅最大的3只股票，将其所有涨跌幅用np.sum()相加，再用np.sort排序；
  3. 再用np.argsort把上一步中的三只股票进行排序并输出股票代码。

小结

其实仔细看一下numpy的说明章节内容并不多，思路也并不是很清晰；这跟我之前学习过的一些机器学习课程是一样的，讲师自己也很难接地气地完整的表达清楚他们想教的东西。

以后在numpy的使用中，估计还是少不了文档支持。
